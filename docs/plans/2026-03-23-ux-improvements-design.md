# UX Improvements — Navigation, Interactivity, Connectivity

**Date:** 2026-03-23
**Status:** Approved

## 1. Navigation
- Sidebar: 6 fixed items (Home, Trilhas, Quiz, Explorar, Ferramentas, Buscar) — zero expandable submenus
- Explore Hub page (`/explore`): grid of 8 categories, search, progress per category
- Tools Hub page (`/tools`): same pattern for tools

## 2. Interactivity
- FlowDiagram: SVG sequence diagrams with animated steps, click to expand
- InteractiveLayerMap: 6-layer payment stack, clickable features
- ComparisonTable: toggle comparisons, sortable, highlight differences

## 3. Connectivity
- content-map.ts: relations for all 77+ pages
- RelatedContent component: 3-4 cards at page footer
- PrerequisiteBanner: recommended prereqs based on progress
- Smart Next: personalized next page recommendation

## 4. Coherence
- Progress Map: skill-tree visualization of all pages
- Daily Challenge: 1 question/day on Home
- Smart Next: algorithmic next-page recommendation
- Onboarding overlay for first-time visitors

## 5. Content Quality
- "Dados do Mercado" block per page (real BR benchmarks)
- "Na Prática" block (anonymized real scenarios)
- "Armadilhas Comuns" block (expert-level edge cases)

## 6. Inline Learning
- ThinkBox: provocative questions mid-content
- TryItYourself: inline mini-calculators
- CompareCards: side-by-side trade-off choices
- KeyInsight: non-obvious senior-level insights
